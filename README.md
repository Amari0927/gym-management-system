# gym-management-system
A full-stack web application designed to manage gym memberships, trainers, and client data. The system provides role-based access for administrators, trainers, and members, allowing efficient management of gym operations.

⸻

Features
	•	User authentication (Login & Registration)
	•	Role-based access control (Admin, Trainer, Member)
	•	Member registration and profile management
	•	Trainer information management
	•	Membership plan management
	•	CRUD operations for users and gym data
	•	Secure backend API using Express and MongoDB

⸻

Tech Stack

Frontend
	•	HTML
	•	CSS
	•	JavaScript

Backend
	•	Node.js
	•	Express.js

Database
	•	MongoDB (Mongoose ODM)

⸻

 System Architecture
	•	Frontend handles UI rendering and form submissions.
	•	Backend provides RESTful API endpoints.
	•	MongoDB stores user, trainer, and membership data.
	•	Authentication and authorization middleware protects restricted routes.
